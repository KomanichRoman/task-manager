# AI Task Manager

Веб-приложение для управления задачами с возможностью создания, просмотра, изменения статуса и удаления задач.

## Стек технологий

- **Frontend:** React + Vite  
- **Backend:** Node.js + Express  
- **База данных:** PostgreSQL  
- **Дополнительно:** Python (экспорт задач в CSV)

## Требования для запуска

Перед запуском убедитесь, что на вашем компьютере установлено:

- **Node.js** (v18 или выше)  
- **PostgreSQL** (v14 или выше)  
- **Python** (v3.10 или выше)

## Примечание для Windows
Если при выполнении команд в терминале (PowerShell) возникает ошибка `Невозможно загрузить файл npm.ps1, так как выполнение сценариев отключено`, используйте префикс `cmd /c` перед каждой командой.

**Пример:**

```bash
cmd /c npm install
```

## Установка и запуск

### 1. Распакуйте архив

Распакуйте полученный архив в любую папку на вашем компьютере.  
Откройте эту папку в VS Code.

### 2. Настройка базы данных

1. Создайте пустую базу данныхх, для этого:
   1. Откройте pgAdmin.
   2. В левой панели разверните Servers → PostgreSQL.
   3. Нажмите правой кнопкой на Databases и выберите Create → Database.
   4. В поле Database введите tasklist, остальное оставьте по умолчанию и нажмите Save.

2. Подключитесь к базе tasklist, для этого:
   1. В левой панели разверните Servers → PostgreSQL → Databases.
   2. Нажмите правой кнопкой на tasklist и выберите "Query Tool".
   3. В открывшемся окне нажмите кнопку Open File (Ctrl + O) выберите и выполните SQL-скрипт из папки sql (файл database.sql).

Этот скрипт создаст таблицу tasks со следующими полями:
* id — уникальный идентификатор
* title — название задачи
* description — описание
* status — статус (new / in_progress / done)
* created_at — дата создания

3. В файле backend/src/config/db.js укажите свой пароль пользователя postgres, который вы вводили при установке PostgreSQL:

```javascript
password: 'ваш_пароль'
```

### 3. Запуск бэкенда

Откройте терминал в папке backend (для этого кликните ПКМ по папке и выберите `Open in integrated Terminal`) и выполните команды:

```bash
npm install
node server.js
```

Сервер запустится на порту 5000.

### 4. Запуск фронтенда

Откройте новый терминал в папке frontend (не закрывая бэкенд) и выполните:

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

### 5. Экспорт задач в CSV (Python)

Откройте файл `python/export_tasks.py` в редакторе кода и укажите свой пароль от PostgreSQL в строке:

```python
password: 'ваш_пароль'
```
Далее откройте терминал в папке python и выполните:

```bash
pip install psycopg2-binary
python export_tasks.py
```

После выполнения в папке python появится файл tasks_export.csv со списком всех задач в формате csv.

## Использование
После запуска фронтенда и бэкенда:
1. Откройте http://localhost:5173 в браузере.
2. Введите название и описание задачи → нажмите "Добавить задачу".
3. Задача появится в таблице.
4. Измените статус через выпадающий список:
- 🆕 Новая
- ⏳ В процессе
- ✅ Выполнена
5. Удалите задачу кнопкой "Удалить".
